# MySQL Portfolio

This repository contains SQL queries written in MySQL to analyze real-world datasets.

This repository contains SQL queries written in MySQL to analyze three datasets:

- `orders` – customer order and revenue information  
- `customers` – customer details  
- `employeedata` – employee information and attendance

The project demonstrates SQL concepts including joins, aggregations, window functions, subqueries, date functions and CTE's.

---

## Topics Covered
✔ SELECT and WHERE  
✔ JOINS  
✔ GROUP BY and HAVING  
✔ Window Functions  
✔ Date Functions  
✔ Subqueries  
✔ CTEs

*** All queries are available inside the `.sql` files in the `/queries` folder. ***
